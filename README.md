# homelab
My personal homelab documenting networking, server administration, and virtualization experiments using Dell PowerEdge servers while studying for the CCNA.

## My IT / Networking Homelab

This repository documents my journey learning networking and systems administration while studying for the CCNA.

## Hardware

- Cisco Catalyst 2960-S (10G)
- Dell PowerEdge R710
- Dell PowerEdge R720XD (currently troubleshooting)
- Mini PC x2 (Optiplex/Thinkcentre Tiny) as Tailscale nodes
- QNAP 2-Bay NAS (RAID1 – 2x2TB)

## Goals

- Learn networking fundamentals
- Practice CCNA labs
- Build virtualization infrastructure
- Document troubleshooting and configuration
- Improve system administration skills

## Current Projects

- Diagnosing R720XD boot issues
- Setting up R710 as a Domain Controller with Active Directory
- Designing and 3D printing a DDR3 holder for server RAM

## Lab Environment

The homelab is used for testing networking concepts, virtualization platforms, and system administration tools. It allows experimentation without impacting production systems.
